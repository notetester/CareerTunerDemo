# CareerTuner Second Brain Viewer

이 폴더는 비공개 Obsidian vault의 Graphify/LLM Wiki 운영 구조를 공개 가능한 형태로 재구성한 GitHub Pages 뷰어다.

- 원본 raw 자료와 private 판단은 비공개 Obsidian submodule에 유지한다.
- 이 공개본은 `graphify-out/public/`의 sanitized graph만 사용한다.
- API key, 계정, endpoint, raw model output, local path는 포함하지 않는다.

Live URL: <https://notetester.github.io/CareerTunerDemo/Obsidian/SecondBrain/>
