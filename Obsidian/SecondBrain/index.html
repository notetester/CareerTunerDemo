Agent Protocol

3-layer query rule

  1. Graph query graph.json, GRAPH_REPORT, wiki/index에서 관련 hub와 후보 문서를 먼저 찾습니다.
  2. Obsidian wiki search concept, system, operation page를 읽어 결정과 synthesis를 확인합니다.
  3. Raw/source reading 구현 변경이나 정본 확인이 필요할 때만 raw와 source tree로 내려갑니다.
