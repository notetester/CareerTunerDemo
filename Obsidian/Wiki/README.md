# CareerTuner Public Wiki

`docs/obsidian-vault/wiki/`와 `graphify-out/GRAPH_REPORT.md`의 공개 가능한 합성 지식을 정적 문서 탐색기로 투영한다.

현재 공개본은 Karpathy의 LLM Wiki seed를 CareerTuner에 적용한 architecture, schema, compounding knowledge, index/log, ingest/query/lint, search tooling, Obsidian workflow를 포함해 20개 문서와 84개 section으로 구성한다.

Live URL: <https://notetester.github.io/CareerTunerDemo/Obsidian/Wiki/>

- URL hash는 문서 식별자다. 예: `#systems/agent-memory-protocol`
- 원문 기반 문서에는 공개 source provenance를 표시한다.
- 원본 private vault를 그대로 복사하지 않고 공개 가능한 설명만 구조화한다.
- API 키, credential, 로컬 절대경로, private raw output은 포함하지 않는다.
