# CareerTuner Public Knowledge Map

이 폴더는 공개 GitHub Pages 데모에서만 제공하는 CareerTuner 심사용 지식맵이다.

- 원본 Obsidian vault는 비공개 저장소에 유지한다.
- 이 공개본에는 내부 노트, 원자료, raw output, 비공개 링크를 포함하지 않는다.
- 공개 가능한 제품 구조, 기술 축, AI/ML 범위, 배포 산출물만 정적 HTML로 재구성한다.

Live URL: <https://notetester.github.io/CareerTunerDemo/Obsidian/>
